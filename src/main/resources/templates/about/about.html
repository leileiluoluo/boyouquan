关于本站

您好，欢迎来到博友圈！

在这个快消的年代，还有多少人仍在坚持更新与关注独立博客？当你躺在床上，一段段短视频刷过后，它们会在你的记忆中留下什么？我这个从 Web 时代过来的「老网民」还是觉得 Web 时代诞生的独立博客的内容更能经得起时间的考验，对我们的思想与生活也更有助益。这并不是媒介不同所引起的，而是写独立博客需要更长时间的思考与沉淀，所以内容也就更有深度一些。

博友圈的建站初衷即源于对独立博客坚守着的敬畏，秉持「不能让好的独立博客无人问津，更不能让它们在时光的大河中就此消亡」。博友圈致力于建立一个博客人的专属朋友圈，提供一个免费的平台，让各个领域的优质独立博客博主连接起来，让博主间有一个可以发现彼此、关注彼此、助力彼此的共同圈子。

若您想推荐自己的博客或者他人的博客被本站收录，可以通过邮件与我联系！（需提供 RSS 地址、博主邮箱和博客简介）

此外，博友圈的各项功能还在不断添加与完善中，如您有好的意见或建议，也可以通过邮件或 GitHub 与我联系！

联系方式如下：

  • 邮箱：contact@boyouquan.com（提交博客、提出意见或建议等）
  • GitHub：github.com/olzhy/boyouquan（仅关注项目本身的问题，如：功能特性、Bug 等，通过提交 Issues 的方式传达）

另外，您可以本着自愿的原则，将本站链接添加到您博客的适当位置，以让更多的博客人发现我们这个小众的圈子！
